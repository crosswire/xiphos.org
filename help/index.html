Help

How we can help you

There are several ways to find help for Xiphos-related problems:

  • Look at the documentation that comes with Xiphos. It is much improved.
  • Join a Xiphos mailing list and ask questions there. If you are a normal user, join the Xiphos User list. You might also consider joining the Xiphos Developer list, if you think it might be more helpful for your problem.
    Neither of the lists have high traffic.
  • Ask someone on IRC at Freenode, channel #xiphos.

How you can help us

There are several ways to help the Xiphos development team:

  • Prayer.
  • Join the development team, whether you are a programmer, artist, writer, translator, packager, or organizer.
  • Bug reports and feedback in general to improve quality and to get to know what the average user wants. Use the GitHub issue tracker or the mailing lists.
